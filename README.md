## Run Locally

Clone the project

```bash
  git clone https://github.com/MarceeloDominguez/menu_animation.git
```

Go to the project directory

```bash
  cd menu_animation
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
